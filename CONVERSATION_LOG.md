## 六、关键技术细节

### 1. 各平台输入框适配

| 平台 | 输入框选择器 | 写入方式 |
|------|--------------|----------|
| ChatGPT | `#prompt-textarea`（ProseMirror） | execCommand('insertText') + InputEvent |
| Claude | `div.ProseMirror[contenteditable]` | 同 ChatGPT |
| Gemini | `rich-textarea` Web Component | 穿透 Shadow DOM |
| DeepSeek | `textarea`（原生） | nativeInputValueSetter + input event |
| 豆包 | textarea 或 contenteditable | 运行时检测 |

### 2. 提示词优化方案
- **主方案：** Background 创建隐藏 tab → Content Script 注入 meta prompt → 等待 AI 回复 → 提取结果 → 关闭 tab
- **Fallback：** 本地规则引擎（角色定义补充、输出格式补充、结构化包装）

### 3. 浮动按钮
- 使用 Shadow DOM 隔离样式，避免与宿主页面冲突
- MutationObserver 监听 SPA 路由切换，自动重新挂载
- 状态：idle / loading / success / error

### 4. Popup 跨页消息流
```
Popup 点击插入
  → chrome.runtime.sendMessage({ type: 'INSERT_TEMPLATE' })
  → Background 接收
  → 检查活动 tab 是否为支持的 AI 平台
  → 已加载 content script：直接转发消息
  → 未加载：用 chrome.scripting.executeScript 注入 fn 直接写入
```

## 七、开发过程中的关键问题与解决

### 问题 1：PowerShell 执行策略阻止 npm
- **现象：** `npm.ps1 cannot be loaded because running scripts is disabled`
- **解决：** `Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass`

### 问题 2：node_modules 安装冲突
- **现象：** 两次 npm install 并行导致目录状态损坏
- **解决：** `cmd /c rmdir /s /q node_modules` 强制清理后重装

### 问题 3：Vite 找不到 manifest 资源
- **现象：** `Could not load manifest asset "src/assets/icons/icon16.png"`
- **解决：** 用 Node.js 内置 zlib 生成纯色 PNG 占位图标

### 问题 4：Popup 的"插入"按钮没反应
- **原因：** 用户在非 AI 平台页面点击，content script 未加载
- **解决：**
  - 检测当前 tab URL，不在支持平台时 fallback 到剪贴板
  - 加 `scripting` 权限 + 动态注入函数兜底
  - 按钮加状态反馈："已插入" / "已复制(请打开AI页面)"

### 问题 5：英文界面下 tags 还显示中文
- **原因：** tags 字段是 `string[]`，硬编码中文
- **解决：** 改为 `{ zh: string; en: string }[]`，渲染时按 locale 取值

### 问题 6：英文界面下插入的模板内容是中文
- **原因：** prompt 字段也是单一字符串
- **解决：** 改为 `{ zh: string; en: string }`，所有 62 个模板补充英文版

### 问题 7：头部语言切换不持久化
- **原因：** 头部按钮只更新 React state，没调用 saveSettings
- **解决：** 加 handleLocaleToggle 同时保存到 chrome.storage.local

## 九、使用说明

### 开发命令
```bash
cd "D:\Program Files\Zozo\promptpro"
# 必要时绕过执行策略
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass

npm run dev      # 开发模式（HMR 热更新）
npm run build    # 生产构建，输出到 dist/
```

### Chrome 加载方式
1. 浏览器地址栏输入 `chrome://extensions/`
2. 开启右上角"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 `D:\Program Files\Zozo\promptpro\dist` 目录
5. 修改代码后点击插件卡片右下角的刷新按钮

### 使用流程
1. 打开任意支持的 AI 平台（ChatGPT、Claude、Gemini、DeepSeek、豆包）
2. **优化提示词：** 在输入框输入文本 → 点击右上角紫色浮动按钮 → 自动优化并填回
3. **插入模板：** 点击浏览器右上角 PromptPro 图标 → 选择分类/搜索 → 点"插入"
4. **切换语言：** Popup 顶部 EN/中 按钮，模板内容会随语言切换
